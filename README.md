# react-digital-bazar

Установка:
```
git clone git@github.com:pnikonov/react-digital-bazar.git
cd ./react-digital-bazar
composer install
yarn
yarn run encore production
```

Запуск сервера:
```
php bin/console server:run
```

Проект будет дотупен по адресу http://127.0.0.1:8000/
